# Web

Next.js frontend scaffold for the DApp.

## Installed

- `next`
- `react`
- `wagmi`
- `viem`
- `@tanstack/react-query`

## Run

```bash
npm run dev
```

## Suggested next steps

- Add wallet provider wiring in `src/lib/web3`
- Add shared UI in `src/components`
- Add app hooks in `src/hooks`
- Add ABI and address exports in `src/contracts`
- Replace `src/app/page.tsx` with your actual app shell

## Notes

- The nested `.git` created by `create-next-app` was removed on purpose.
- Keep this folder focused on frontend only.
