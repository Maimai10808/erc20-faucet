# Hardhat

Hardhat workspace for contracts, deployment modules, and scripts.

## Installed by init

- `hardhat`
- `@nomicfoundation/hardhat-toolbox-viem`
- `@nomicfoundation/hardhat-ignition`
- `viem`

## Current dirs

- `contracts`
- `ignition/modules`
- `scripts`
- `test`

## Run

```bash
npx hardhat compile
npx hardhat test
```

## Notes

- Default sample contract, tests, and scripts were removed.
- Add your own contract files under `contracts`.
